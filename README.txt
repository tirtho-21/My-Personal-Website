# Image Requirements for Personal Website

This folder should contain all the images for your personal website. Below is a list of required images and their recommended dimensions:

## Required Images:

1. **Logo**: `logo.png` (200x50px) - Preferably with transparent background
2. **Welcome Background**: `welcome-bg.jpg` (1920x1080px) - Hero section background
3. **Featured Image**: `featured-image.jpg` (1200x600px) - Main featured image
4. **About Me Image**: `about-me.jpg` (600x800px) - Profile picture for About page
5. **CV Preview**: `cv-preview.jpg` (800x1000px) - Preview of your CV
6. **CTA Background**: `cta-bg.jpg` (1920x1080px) - Call to action section background

## Carousel Images:
- `carousel-1.jpg` through `carousel-4.jpg` (800x600px) - For the main carousel

## Gallery Images:
- `gallery-1.jpg` through `gallery-8.jpg` (800x600px) - For the gallery grid and gallery carousel

## Testimonial Images:
- `testimonial-1.jpg` through `testimonial-3.jpg` (200x200px) - Client/testimonial profile pictures

## Blog Images:
- `blog-1.jpg` through `blog-3.jpg` (800x400px) - Blog post thumbnails

## Image Optimization:
- All images should be optimized for web (compressed) to ensure fast loading times
- Use JPG format for photographs and PNG for images requiring transparency
- Keep file sizes under 300KB when possible

## Placeholder Images:
If you need placeholder images during development, you can use services like:
- [Unsplash](https://unsplash.com/) - High-quality free images
- [Pexels](https://www.pexels.com/) - Free stock photos
- [Lorem Picsum](https://picsum.photos/) - Easy placeholder images (e.g., https://picsum.photos/800/600)
